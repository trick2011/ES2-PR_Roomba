  Project information


 More and more robots are deployed to take over simple tasks. Domestic example, the iRobot Roomba robotic vacuum cleaner will be
 deployed to the to keep house dust. The iRobot Roomba has an open interface allowing control can be taken over by the robot.

 The project is called AVOIR, which is a reference to alternative object -oriented programmed Irobot Roombacontroller. The aim of
 this project is through reverse engineering functionality to identify the iRobot Roomba and entrusting them to an alternative iRobot
 Roombacontroller.

 Our mission is:
 " Provide a working prototype , a Roombacontroller, which can be used to drive along an iRobot Roomba robotic vacuum cleaner."
